# Terraform AWS VPC Lab

This repository contains a Terraform-based AWS VPC build created entirely from the macOS terminal.

## What This Builds
- Custom VPC (10.0.0.0/16)
- Public subnet
- Internet Gateway
- Route table + association

## Tools Used
- Terraform
- AWS CLI
- macOS Terminal

## How to Use
```bash
terraform init
terraform plan
terraform apply
